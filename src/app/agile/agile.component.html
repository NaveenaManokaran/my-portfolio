Agile Resource Tool - April to December 2018
The nature of this work is confidential

The challenge: Our client Cisco approached to develop an interactive and smooth front end experience for the manager and the top hierarchy people. The project required a website with the enhanced features remaining fast to use by everyone which is not present in the existing process.

The solution: To reduce the overall estimated time and cost, I have used HTML5, CSS3 and Bootstrap. These gave responsive layout which has been customized based on the design with good visual effects. Charts have been designed to show the date details in more simple way and quick to understand.
Cisco provided me with the spreadsheet for each layout and relied on my experience to create a new, friendlier desktop website including all features adding some enhanced functionality.

The result: The agile resource tool helps to create a new Project and Manage Team Members allocation, Team work capacity and Get suggestions for it. Manage project has features like Edit, Clone, Delete your project, Export Project suggestions, View Version History.
Functionalities has been provided with some formulas to calculate the team planning. History is shown by comparing two different versions using a pie chart to provide more realistic and comfortable understanding of the data.
